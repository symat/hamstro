# hamstro

Hamstro estas malgranda portebla ŝatokupa komputilo bazita sur 8-bita AVR microcontroller. Ĝi estas malgranda gameboy, kiu povas esti konektita al dokstacio, kiu ebligas plilongigitajn funkciojn transformante la Hamstro-tabulon al plena, memstara komputilo.



La ĉefa lingvo de la projekto estas Esperanto (nur por mia diro, por lerni kaj praktiki la lingvon).
